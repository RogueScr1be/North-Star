## Available Commands

### Pinchtab Server (Background)
```bash
~/bin/pinchtab                         # Main server (port 9867)
~/bin/pinchtab bridge                  # Single-instance mode
```

### CLI Commands (Foreground)
```bash
# Navigation
pinchtab nav https://example.com

# Inspection
pinchtab snap -i                       # Interactive elements
pinchtab snap -i -c                    # With coordinates
pinchtab text                          # All text content

# Interaction
pinchtab click e5                      # Click element
pinchtab fill e3 "value"               # Fill input
pinchtab press e7 Enter                # Press key
pinchtab type e3 "text"                # Type text
```

### HTTP API (Curl)
```bash
# Instance management
curl http://localhost:9867/instances
curl -X POST http://localhost:9867/instances/launch

# Tab management
curl http://localhost:9867/instances/{id}/tabs/open

# Snapshot & interaction
curl http://localhost:9867/tabs/{id}/snapshot
curl http://localhost:9867/tabs/{id}/text
curl -X POST http://localhost:9867/tabs/{id}/action
```

## Troubleshooting

### "Connection refused" on port 9867
```bash
# Check if Pinchtab is running
curl http://localhost:9867/health

# If not running, start it
~/bin/pinchtab

# If port in use, kill existing process
lsof -i :9867 | awk 'NR==2 {print $2}' | xargs kill
```

### "No elements found" in snapshot
```bash
# Use filter to get interactive elements only
pinchtab snap -i

# Not all elements are interactive (filter=interactive)
# Use snap -i -c to see coordinates
```

### "Element ref invalid"
```bash
# Element refs are only valid for current page state
# Always get fresh snapshot before clicking
pinchtab snap -i -c
pinchtab click e5  # e5 is from CURRENT snapshot only
```

### Integration with Testing
Create test scripts that use Pinchtab API:

```bash
#!/bin/bash
# Automated testing with Pinchtab

PINCHTAB_API="http://localhost:9867"
NS_URL="http://localhost:3000/constellation"

# Launch instance
INST=$(curl -s -X POST $PINCHTAB_API/instances/launch \
  -d '{"name":"test","mode":"headless"}' | jq -r '.id')

# Navigate
curl -X POST $PINCHTAB_API/instances/$INST/tabs/open \
  -d "{\"url\":\"$NS_URL\"}" | jq -r '.tabId' > $TAB

# Get snapshot (text-based)
curl $PINCHTAB_API/tabs/$TAB/snapshot?filter=interactive

# Cleanup
curl -X DELETE $PINCHTAB_API/instances/$INST
```
