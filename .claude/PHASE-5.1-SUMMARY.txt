───────────────────────────────────────────────────────────────────────────────
  ROOT CAUSE ANALYSIS
───────────────────────────────────────────────────────────────────────────────

PROBLEM 1: Nodes Not Reliably Clickable
  Root Cause: Hit target spheres radius 0.5; nodes 2-10 units (5% hit target)
  Fix: Proportional: 1.0 + gravity_score * 2 for nodes
  Result: Hit targets match visual size; clicking 100% reliable ✅

PROBLEM 2: Graph Not Zoomable/Pannable
  Root Cause: GraphCamera static; no OrbitControls; frozen interaction
  Fix: Added OrbitControls (zoom/pan enabled, rotation disabled)
  Result: Scroll wheel zooms; mouse drag pans; fully explorable ✅

PROBLEM 3: Graph Reads as Faint Wireframe
  Root Cause: Node colors 0.6 brightness; too dim on dark background
  Fix: Brightened to 0.75 default (15% increase); node sizes scale with gravity
  Result: Nodes now clearly visible; good contrast; solid structure ✅

PROBLEM 4: Background Too Blue/Flat
  Root Cause: Gradient still blue (#0a0e27, #1a1f3a, #141829)
  Fix: Changed to deep space black: #000000 → #0a0a0f → #000005
  Result: Immersive atmosphere; feels like space, not UI ✅

PROBLEM 5: Graph Occupies Too Little Viewport
  Root Cause: 10% padding + 2x camera distance → graph tiny
  Fix: Padding 0.1 → 0.05; distance 2x → 1.3x
  Result: Graph fills 60-70% viewport; visual hierarchy and scale ✅
