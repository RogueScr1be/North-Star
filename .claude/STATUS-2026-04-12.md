## Key Learnings

1. **React Hook Dependency Management:** State variables set inside useEffect should not be in the dependency array to avoid redundant re-runs.

2. **OpenAI API Differences:** Chat Completions API (vs legacy Messages) uses different parameter names. Always check SDK documentation for exact endpoint contracts.

3. **Kill Switch Patterns:** Environment variable-based feature flags enable <5 minute rollbacks without code changes.

4. **Measurement Before Scaling:** Phase 6.2 decision gate ensures data-driven decisions about default shifts (not gut feel).

5. **Streaming Implementation:** Native OpenAI streaming (stream=true) integrates naturally without additional complexity.
